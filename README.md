# Car Rental Company Website

A simple car rental website built with **HTML, CSS, JavaScript, and PHP**. It allows customers to browse and rent cars while admins can manage cars, reservations, and reports.

---

## Features

- **Customer Dashboard:** Browse available cars, rent cars, view reservations.
- **Admin Dashboard:** Add, update, delete cars, manage reservations, generate reports.
- **Advanced Search:** Filter cars by type, brand, or availability.
- **Authentication:** User registration, login, logout.
- **Reports:** Admin can view rental and reservation reports.
- **Manual:** Step-by-step user guide included (`manual.html`).

---

## Pages and Files

### Customer & Admin Pages
- `index.php` — Home / Landing page  
- `dashboard.php` — Customer dashboard  
- `admin_dashboard.php` & `Admin_Dashboard.css` — Admin dashboard  
- `Customer_Dashboard.css` — Customer dashboard styling  

### Car Management
- `add_car.php` & `add_car.css` — Add new cars  
- `update_car.php` & `update_car.css` — Update existing cars  
- `delete_car.php` & `delete_car.css` — Delete cars  
- `rent_car.php` & `rent_car.css` — Rent a car  

### Search & Reports
- `advanced_search.php` & `advanced_search.css` — Search cars with filters  
- `reports.php` & `reports.css` — Admin reports page  
- `reservations.php` & `reservations.css` — View all reservations  

### Authentication
- `register.php` & `register.css` — User registration  
- `logout.php` — Log out  
- `header.php` — Common header for all pages  

### Backend / Database
- `db.php` & `db_connection.php` — Database connection files  
- `process_reservation.php` — Handles rental/reservation logic  

### Extras
- `help.css` — Help page styling  
- `manual.html` — User manual  
- `Tables.txt` — Database tables  
- `Final Project Database.docx` — Database structure  
- `x.txt` — Miscellaneous file  
- `mod.css` — Additional CSS  

---

## Usage

1. Clone the repository:
   ```bash
   git clone <repository-url>

2. Move the files to your PHP server directory (e.g., XAMPP htdocs).

3. Import the database using Final Project Database.docx or Tables.txt.

4. Open index.php in your browser.

## Technologies

- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
